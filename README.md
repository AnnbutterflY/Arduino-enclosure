# 前言 Forward
本次课题是为Arduino设计外壳，制作要求为

**1、** 结合 3D 打印与激光切割技术；

**2、** 禁止使用胶水，仅可使用螺丝或机械连接；

**3、** 设计一个具有未来合理性的产品（Speculative Product），并为其制作外壳，推想该设计从构建一个可发生的未来开始。通过“未来世界”的设定，设计可以摆脱当下限制，探索替代思维。

## 由此，我的世界构想为:

**未来法律设定：** 所有家庭和公共空间不得使用一次性电子产品，所有硬件设备必须采用“可生长、可修复”的模块化设计，通过环境引导其功能发展。人们像培育植物一样“养”设备，而非简单购买固定功能的产品.

**社会形态与关键事件：** 地球经历工业体系崩塌和环境严重破坏的关键事件后，人类社会转向技术与生态共生的模式。社区不再依赖大型工厂生产日常设备，而是以家庭和社区为单位，通过“培养”技术植物满足生产和生活需求。社会形态呈现去中心化、生态化与模块化协作特征。

**整体环境与视觉风格：** 城市与废墟交融，建筑表面、屋顶、墙面遍布发光的电子藤蔓和生长的设备花朵。视觉风格融合科技与有机生物形态，光线柔和、呼吸式光源贯穿环境，半废墟、半生机的未来景观。

**生活方式、日常行为与核心需求:** 人们的生活不再围绕购买设备，而是通过培育和引导设备成长来满足生活需求。日常行为像照顾植物一样：调节光照、湿度和养分，引导设备长出不同功能模块。核心需求从“拥有设备”转向“培养、共生与管理设备成长”。
